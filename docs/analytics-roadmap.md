# Analytics and Admin Roadmap

## Phase 1
- Add a lightweight analytics configuration layer.
- Track portfolio views and resume downloads through a backend endpoint.
- Prepare the portfolio for Supabase or Firebase integration.

## Phase 2
- Add an admin dashboard for editing content.
- Store contact form submissions and download events.
- Display visitor summaries and recruiter activity signals.

## Phase 3
- Connect an AI assistant over the same content store.
- Show recruiter-focused analytics and engagement insights.
- Expand into a richer Business Central technical consultant portal.
